# stjerm Terminal Emulator

## Requirements

pkgconfig, gtk2, vte

### Ubuntu Requirements

If you're compiling from source on Ubuntu, you'll need:

    sudo apt-get install build-essential automake libglib2.0-dev libvte-dev

## Installation

    ./autogen.sh
    make
    sudo make install


## Running

Simply run:
    stjerm

and all available options will be listed.
